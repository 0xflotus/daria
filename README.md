# Usage
`daria`

## Specify date
`daria 2019-01-02`

## Format date
`daria -f "Today is %s"`

### formatting options

|Identifier|Description|
|-|-|
|%D|Mars Day|
|%H|Mars Hour|
|%M|Mars Minute|
|%N|Mars Month|
|%S|Mars Second|
|%Y|Mars Year|
|%m|Mars Month Name|
|%s|Mars Sol Name|
